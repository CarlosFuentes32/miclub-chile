# Plan PILOTO COMERCIAL

El plan debe ser configurable por Super Admin.

Campos:

- Código interno: `PILOTO_COMERCIAL`.
- Precio: pendiente de aprobación del dueño.
- Duración: pendiente de aprobación.
- Trial: configurable.
- Gracia: configurable.
- Límite máximo: 5 comercios piloto.
- Soporte: cercano/prioritario durante piloto.
- Cobro: transferencia manual.
- Renovación: manual hasta integrar proveedor.

No definir valores finales sin aprobación comercial.
