# Política de soporte piloto

Canal oficial: pendiente de confirmación comercial.

Horario sugerido:

- Lunes a viernes horario comercial.
- Incidentes críticos: atención prioritaria.

Prioridades:

- Crítico: sistema no disponible, canje imposible, pérdida de datos.
- Alto: login bloqueado, comercio no puede operar.
- Medio: errores funcionales no bloqueantes.
- Bajo: consultas/mejoras.

SLA interno piloto:

- Crítico: primera respuesta < 1 hora hábil.
- Alto: < 4 horas hábiles.
- Medio: < 1 día hábil.
- Bajo: según planificación.

Todo caso debe registrarse como ticket o incidente.
