# Alcance sugerido para la Fase 5

1. Crear relaciones entre cajeros, comercios y sucursales.
2. Implementar endpoints reales e idempotentes para búsqueda, transacciones, canjes y anulaciones.
3. Emitir y validar QR firmados, rotativos y con vencimiento.
4. Definir reglas reales de progreso y desbloqueo dentro del motor de fidelización.
5. Persistir auditoría inmutable con actor, motivo, dispositivo y marcas de tiempo.
6. Añadir control de concurrencia para impedir dobles canjes o transacciones duplicadas.
7. Incorporar pruebas de cámara, accesibilidad, dispositivos físicos y operación con conectividad inestable.

Campañas, reportes avanzados, geolocalización y beneficios cruzados permanecen fuera del alcance.
