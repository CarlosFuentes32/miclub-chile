# Límite tributario

Registrar un pago no equivale a emitir boleta, factura ni documento tributario.

Proceso actual:

- El sistema registra pagos y estados comerciales.
- La emisión tributaria, si aplica, queda como proceso administrativo externo.
- No se generan documentos tributarios falsos.

La integración tributaria futura debe definirse legal y contablemente antes de automatizarse.
