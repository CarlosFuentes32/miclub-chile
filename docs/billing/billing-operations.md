# Operación de billing

Para pilotos:

1. Crear plan configurable.
2. Asignar plan al comercio.
3. Registrar transferencia como pago pendiente.
4. Revisar evidencia.
5. Aprobar o rechazar.
6. Verificar historial financiero.
7. Ejecutar recordatorios si corresponde.

Resend staging debe configurarse para validar correos reales. Si no está configurado, los correos quedan omitidos de forma segura.
