# Suscripciones

Cada comercio tiene una suscripción principal. Estados:

- `draft`
- `trialing`
- `active`
- `grace_period`
- `past_due`
- `suspended`
- `cancelled`
- `expired`

La suscripción guarda fechas de inicio, trial, periodo actual, próximo vencimiento, gracia, suspensión, reactivación, cancelación, precio acordado, descuento, método de pago y metadata segura.
