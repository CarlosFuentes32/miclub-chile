# Checklist de lanzamiento comercial

- [ ] Crear comercio demo.
- [ ] Crear programa de fidelización.
- [ ] Crear usuario cajero.
- [ ] Imprimir QR de inscripción.
- [ ] Probar registro de cliente.
- [ ] Probar transacción.
- [ ] Probar avance y generación de recompensa.
- [ ] Probar canje.
- [ ] Revisar dashboard del comercio.
- [ ] Revisar landing en celular y computador.
- [ ] Preparar discurso de venta.
- [ ] Confirmar precios y condiciones del piloto.
- [ ] Tener una alternativa manual si falla internet o cámara.
