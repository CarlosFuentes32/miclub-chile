# Permisos del rol SUPPORT

Permitido:

- Buscar usuarios, comercios, tickets, request IDs, incidentes, transacciones y recompensas.
- Ver datos enmascarados y fichas 360.
- Crear y actualizar tickets.
- Agregar notas internas.
- Enviar recuperación de contraseña con ticket activo.
- Revocar sesiones con ticket activo.
- Desbloquear cuenta con ticket activo.

Prohibido:

- Eliminar usuarios o comercios.
- Cambiar roles.
- Cambiar plan o registrar pagos.
- Ejecutar backups, restores o rollbacks.
- Exportar bases completas.
- Ver secretos, contraseñas o tokens.
- Modificar puntos, recompensas o transacciones directamente.
