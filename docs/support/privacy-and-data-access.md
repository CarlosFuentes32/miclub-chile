# Privacidad y acceso justificado

El soporte debe indicar motivo para consultar datos sensibles.

Datos enmascarados por defecto:

- Email.
- Teléfono.
- RUT.
- IP.
- User-Agent.

No se muestran contraseñas, refresh tokens, API keys ni valores secretos.

Las notas internas rechazan patrones evidentes de secretos.
