# Herramientas operativas de soporte

Herramientas habilitadas:

- Enviar recuperación de contraseña.
- Revocar una sesión o todas las sesiones.
- Desbloquear cuenta.
- Consultar estado del sistema desde panel existente.

Requisitos:

- Ticket activo.
- Motivo explícito.
- Auditoría.
- Rate limiting global.
- Sin exposición de tokens ni contraseñas.
