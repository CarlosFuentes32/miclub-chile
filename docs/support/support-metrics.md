# Métricas de soporte

El dashboard muestra métricas reales existentes:

- Tickets abiertos.
- Tickets críticos.
- Tickets vencidos.
- Incidentes activos.
- Usuarios bloqueados.
- Comercios suspendidos.
- Errores recientes.
- Sesiones revocadas.
- Tickets asignados al agente.

Promedios quedan `null` hasta tener suficientes tickets resueltos.
