# Runbook: usuario no puede iniciar sesión

1. Crear o seleccionar ticket activo.
2. Buscar usuario con motivo.
3. Revisar estado, bloqueo, eliminación lógica y sesiones.
4. Revisar auditoría y errores relacionados.
5. Si corresponde, enviar recuperación o desbloquear.
6. Registrar nota interna y cerrar o escalar.
