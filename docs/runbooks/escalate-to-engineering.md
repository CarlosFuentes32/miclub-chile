# Runbook: escalar a ingeniería

Escalar cuando exista:

- Error 5xx reproducible.
- Inconsistencia entre transacción/recompensa/ciclo.
- Falla de health ready.
- Incidente activo.
- Request ID con error crítico.

Incluir ticket, request ID, usuario/comercio enmascarado, pasos, impacto y evidencia.
