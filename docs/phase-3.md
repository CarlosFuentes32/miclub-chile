# Alcance sugerido para la Fase 3

1. Vincular usuarios con comercios y sucursales mediante membresías explícitas.
2. Implementar administración segura de usuarios, invitaciones y recuperación de contraseña.
3. Definir el modelo de programa de fidelización antes de crear lógica de puntos.
4. Diseñar navegación y layouts finales por portal.
5. Añadir pruebas unitarias, integración de autenticación y pruebas end-to-end por rol.
6. Incorporar rate limiting, auditoría de accesos y política de bloqueo por intentos fallidos.
7. Definir correo transaccional, verificación de email/teléfono y consentimiento de datos.
8. Preparar CI/CD, HTTPS, secretos administrados, observabilidad y despliegue.

Los ciclos, recompensas y transacciones continúan intencionalmente sin implementación.
